#

## 專案說明
以聊天室為主題的公開討論網站。

### 設計概念
希望給朋友或家人有一個能夠互相分享資訊、共同討論話題的公開空間。

#
### 使用環境
- `Python3`。

### 使用技術
- 以 `Django` 框架架設開發。
- 使用 `Tailwind CSS` 框架為主，以進行切版。


### 使用套件
- `Django`
- `django-cors-headers`
- `djangorestframework`
- `mysqlclient`
- `Pillow`
- `PyMySQL`
- `whitenoise`
- `tailwind`
